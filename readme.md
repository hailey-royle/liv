
# Liv!

Liv is a text editor I have made for my personal idiosyncrasys and fun. 
It is currently working!  

# Usage

Liv has two modes, insert and command.  
On start, liv is in command mode.  
Enter insert mode by pressing 'i', exit by pressing 'escape'.  

## Commands

All commands follow the pattern of, [command - count - motion]  

### implemented

- q - quit
- w - write
- i - insert
- a - append

- *n* - do thing x times

- h - cursor left
- l - cursor right
- j - line down
- k - line up
- b - word next
- e - word previous
- t - file start
- z - file end
